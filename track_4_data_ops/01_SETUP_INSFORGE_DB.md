# InsForge Database Setup

> [!TIP]
> **Executive Summary:** Run these commands to get the Postgres DB online.

## Boilerplate
1. Log into InsForge Dashboard.
2. Create a new project called `maintenance-agent`.
3. Go to the Database tab and run the SQL provided in `02_SCHEMA_DESIGN.md`.
4. Copy the connection string and paste it into the shared Slack/Discord so everyone can put it in their `.env` files.
