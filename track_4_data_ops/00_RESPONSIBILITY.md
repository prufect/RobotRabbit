# Track 4 Responsibility: Data & Ops Manager

> [!IMPORTANT]
> **Executive Summary:** You are the glue that holds the hackathon together. You own the InsForge Database, the Storage Bucket, the Replicas background agents, and the GitHub merging process. 

## Core Responsibilities
1. Setup the InsForge Postgres Database and execute the schema.
2. Setup the InsForge Storage bucket and expose a signed-URL generator so Track 1 can upload photos.
3. Use Replicas agents to automate boilerplate writing (e.g. generating mock data).
4. Resolve GitHub merge conflicts for your team so they never have to stop coding.

## Dependencies
You do not depend on anyone. Everyone depends on YOU. You must get the Database and Storage up and running in the first 30 minutes.
