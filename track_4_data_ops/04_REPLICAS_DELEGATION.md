# Using Replicas Background Agents

> [!IMPORTANT]
> **Executive Summary:** You are required to use Replicas to impress the judges. 

## How to use Replicas
During the hackathon, log into Replicas and connect it to your GitHub repo.
Issue the following command to the agent:

> *"Hey Replicas, please write a Node.js seed script that connects to our Postgres database using the `pg` library and inserts 10 dummy contractor responses for testing purposes. Put the script in a new `/scripts` folder and open a PR."*

While Replicas does that, you can focus on resolving merge conflicts. Mention this specific workflow in your Devpost submission!
