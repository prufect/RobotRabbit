# Integration Setup Guide

> [!TIP]
> **Executive Summary:** Get your API keys immediately. Do not wait.

## Required API Keys
1. **Twilio:** Sign up for a free Twilio account. Activate the WhatsApp Sandbox. Get your Account SID, Auth Token, and Sandbox Phone Number.
2. **SerpApi:** Sign up for SerpApi for Google Local search. Get your API key.
3. **Telegram (Optional fallback):** Message `@BotFather` on Telegram, create a new bot, and get the HTTP API token.

## Dependencies
```bash
npm install twilio axios
```
